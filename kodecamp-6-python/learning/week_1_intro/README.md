# Week 1: Introduction to Programming and Python Foundations

Welcome to Week 1 of the Python learning track.
This week focuses on the core concepts every beginner needs before moving to advanced topics.

## Learning Goals

By the end of this week, you should be able to:
- Explain what programming is in simple terms.
- Describe why Python is a great first language.
- Install Python and verify your setup.
- Write and run basic Python scripts.
- Use variables and common data types.
- Collect user input and display output.
- Make decisions using `if`, `elif`, and `else`.
- Use comparison and logical operators in conditions.

## Curriculum Outline

1. What is Programming? (`what_is.py`)
2. Why Python? (`why_python.py`)
3. Installing Python and Environment Setup (`installing.py`)
4. Python Syntax and Code Structure (`syntax.py`)
5. Variables and Data Types (`variables.py`)
6. User Input and Output (`input_output.py`)
7. Conditional Statements (`conditional.py`)

## Suggested Learning Flow

1. Read each file in order.
2. Run each script in your terminal using `python3 <filename>`.
3. Modify the examples and test what changes.
4. Complete the exercises at the end of each file.

## Week 1 Practice Tasks

- Write a script that asks for a user's name and age, then prints a greeting.
- Write a script that checks if a number is positive, negative, or zero.
- Write a script that asks for a score and prints grade:
  - `A` for 70 and above
  - `B` for 60-69
  - `C` for 50-59
  - `F` below 50

Keep this week practical: read, run, change, and observe.
