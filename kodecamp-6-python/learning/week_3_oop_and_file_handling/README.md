# Week 3 — Object-Oriented Programming & File Handling

Each topic has its own runnable script. `cd` into this folder, then run any file with `python3 <filename>`.

Suggested order:

| Order | File | Topic |
| --- | --- | --- |
| 1 | `intro_to_oop.py` | What OOP is and why we use it |
| 2 | `classes_and_objects.py` | Defining a class and creating objects |
| 3 | `attributes_and_methods.py` | Data on objects and behavior (methods) |
| 4 | `constructors.py` | `__init__` — set up new objects |
| 5 | `file_modes.py` | `r`, `w`, `a`, and when to use each |
| 6 | `working_with_files.py` | Read, write, and append text files |
| 7 | `working_with_json.py` | JSON basics with `json` module |
| 8 | `python_objects_to_json.py` | Save Python data to JSON |
| 9 | `reading_json_to_python.py` | Load JSON back into Python |
| 10 | `practical_exercises.py` | Exercise prompts |

## Run from this directory

```bash
cd kodecamp-6-python/learning/week_3_oop_and_file_handling
python3 classes_and_objects.py
```

## Run from the repo root

```bash
python3 kodecamp-6-python/learning/week_3_oop_and_file_handling/classes_and_objects.py
```

Some scripts create small files (`notes.txt`, `sample.json`, etc.) in this folder. You can delete them and re-run the scripts anytime.
