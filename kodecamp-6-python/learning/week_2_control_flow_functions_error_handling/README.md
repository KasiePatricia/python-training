# Week 2 — Control flow, functions, and error handling

Each topic has its own runnable script. `cd` into this folder, then run any file with `python3 <filename>`.

Suggested order:

| Order | File | Topic |
| --- | --- | --- |
| 1 | `loops.py` | `for`, `while` |
| 2 | `functions.py` | Defining and calling functions |
| 3 | `function_parameters_and_returns.py` | Parameters, defaults, `*args`, `**kwargs`, returns |
| 4 | `loop_control.py` | `break`, `continue`, `pass` |
| 5 | `nested_loops.py` | Nested loops |
| 6 | `scope.py` | Local vs global variables |
| 7 | `error_handling.py` | `try`, `except`, `finally` |
| 8 | `custom_errors.py` | Raising custom errors |
| 9 | `practical_exercises.py` | Exercise prompts (implement in this or new files) |

## Run from this directory

```bash
cd kodecamp-6-python/learning/week_2_control_flow_functions_error_handling
python3 loops.py
```

## Run from the repo root

```bash
python3 kodecamp-6-python/learning/week_2_control_flow_functions_error_handling/loops.py
```

For the interactive exercise in `practical_exercises.py`, uncomment the `while True` block at the bottom of that file.
